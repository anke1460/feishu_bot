## Installation
Add this line to your application's Gemfile:

```ruby
gem 'feishu_bot'
```

And then execute:
```bash
$ bundle

$ rails feishu_bot:install:migrations
```

add feishu.rb in config/initializers
```
# FeishuBot.configure do |config|
#   config.app_id = ''
#   config.app_secret = ''
# end
```
Or install it yourself as:
```bash
$ gem install feishu_bot
```
